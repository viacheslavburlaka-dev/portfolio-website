# Viacheslav Burlaka — Portfolio

Welcome to my personal portfolio website!

This site showcases who I am, what I'm passionate about, and where I'm headed.
It's built with vanilla HTML, CSS, and JavaScript — no frameworks, no build steps.

## 🌐 Live Site

**https://slava-1f130.web.app**

## 🚀 Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (vanilla)
- **Hosting:** Firebase Hosting
- **Deployment:** GitHub Actions (auto-deploy on push to `main`)

## 📁 Structure

```
portfolio-website/
├── index.html       # The entire site — single page
├── firebase.json    # Firebase hosting config
├── .gitignore       # Files to exclude from git
└── .github/
    └── workflows/
        └── deploy.yml   # Firebase auto-deploy via GitHub Actions
```

## 🛠️ Local Development

Just open `index.html` in your browser:

```bash
open index.html    # macOS
xdg-open index.html  # Linux
```
